Send SMS messages direct from your blog through Messagebird Gateway.

== Description ==

Send SMS messages directly from your Wordpress website. 

It allows all users of your website to send text messages immediately or they can schedule them at a later time or date. Messages will be send through a Gateway by Messagebird.com (a dutch company for payment & text-messages). A free Messagebird account is required.

Users can buy credits from you to send sms messages, an administrator is able to set the price. If you want to attract more registrations for your site you can give new users an xx amount of credits as a gift for their registration. 
For now the plugin is in dutch (NL) only.

== Installation ==

1. Unzip the ZIP file and drop the folder straight into your `wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a free Messagebird account and buy some credits to send text messages
4. From your Wordpress Dashboard, go to Istiecool > IstieSMS > Follow the on-screen cues.
5. In your user admin area you can give users credits to send messages.
